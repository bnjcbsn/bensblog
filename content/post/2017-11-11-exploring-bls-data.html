The BLS data availability

I became curious about this subject following a tweet by a bank with an infographic regarding the November jobs report. I lost track of the original image and elected to use a substitute. I did a web search a day later and decided to make a project of the chart below. (found here) I’m not trying to replicate it perfectly, but learn about BLS data acquisition and including plots in this post. It dawned on that public data such as monthly employment may prove useful for a post as my professional work is entirely not public for sharing. I visited bls.gov (Bureau of Labor Statistics) to try to find out what is available and how to access it. The BLS offers a web interface, an API, and simple text files for accessing the data they routiely publish. To select a launching point, I always start out with a Google query and stackoverflow search. In top results, I discovered a package developed for BLS data access that was created by Kris Eberwein, blscrapeR.
https://cran.r-project.org/web/packages/blscrapeR/vignettes/Employment_and_Unemployment.html

The BLS site does offer text file downloads, and Kris’ package uses that approach, but seems centered on using the API for the site which I will lean away from if possible. I’m still trying to understand any benefits gained from either method over the other.

Time Series Conversion

For reasons discussed by Business Science, I’ll lean on the tidyquant package for impementing time series elements with this analysis.
My conversion to date solution was inspired by this stackoverflow post.

library(tidyquant)

Here, I use zoo::as.yearmon to convert the year and period fields to a more useful date field.

series.01a$date <- as.yearmon(paste(series.01a$year,
                                 substr(series.01a$period, 2, 4),sep="-"))

I filter for the series that I’m seeking, mutate a change value using dplyr::lag, and shed some fields that I don’t need.

fs <- series.01a %>%
filter(series_id=='CES0000000001') %>%
    mutate(chg=value-lag(value)) %>%
    select(date, value, chg)

To conclude, the full code to produce the same chart from start to finish.

library(tidyverse)
library(tidyquant)
url <- "https://download.bls.gov/pub/time.series/ce/ce.data.01a.CurrentSeasAE"
series.01a <- read_tsv(url)

series.01a$date <- as.yearmon(paste(series.01a$year,
                                 substr(series.01a$period, 2, 4),sep="-"))
fs <- series.01a %>%
    filter(series_id=='CES0000000001') %>%
    mutate(chg=value-lag(value)) %>%
    select(date, value, chg)

fs %>%
    mutate(date2=as.Date(date)) %>%
    filter(date > 2007) %>%
    ggplot(aes(x = date2, y = chg)) +
    geom_segment(aes(xend = date2, yend = 0, color = chg)) + 
    labs(title="Monthly jobs growth", subtitle="In thousands", y="", x="") + 
    theme_light() +
    theme(legend.position="none")
